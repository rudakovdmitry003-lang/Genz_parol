# 🔐 Генератор Паролей

**Простой и красивый генератор надёжных паролей** с современным тёмным интерфейсом.

Создавай сильные пароли за один клик — ничего не сохраняется, всё работает **локально**.

### ✨ Возможности

- Три режима генерации: Простой, Сложный и Свои настройки
- Длина пароля от 6 до 50 символов
- Поддержка строчных, заглавных букв, цифр и спецсимволов
- Удобный тёмный дизайн
- Копирование пароля в буфер обмена одним нажатием
- Сборка в один **EXE-файл** (не требует Python)

### 🚀 Скачать готовую версию

Перейди в раздел **[Releases](https://github.com/rudakovdmitry003-lang/Genz_parol/releases)** и скачай `ГенераторПаролей.exe`

### 🛠 Технологии

- Python 3
- Tkinter (графический интерфейс)
- PyInstaller (сборка в exe)
