Генератор Паролей

Современное приложение для генерации надёжных паролей с удобным графическим интерфейсом.

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![License](https://img.shields.io/badge/License-MIT-green)

Описание

Генератор Паролей — это простое, быстрое и безопасное приложение для создания сильных паролей.  
Работает полностью локально (оффлайн), ничего не отправляет в интернет.

Основные возможности

- Три режима генерации:
  - Простой пароль (буквы + цифры)
  - Сложный пароль (всё включено)
  - Полная настройка (выбор типов символов)
- Красивый **тёмный интерфейс** (чёрная тема)
- Всплывающие окна с уведомлениями
- Копирование пароля в буфер обмена одним кликом
- Регулируемая длина пароля (от 6 до 50 символов)
- Оценка надёжности пароля
- Сборка в один **EXE-файл** (не требует установленного Python)

Как запустить

Вариант 1: EXE-файл (рекомендуется)

1. Скачай файл `ГенераторПаролей.exe` из папки `dist`
2. Запусти его (двойной клик)
3. Готово! Ничего устанавливать не нужно.

Вариант 2: Через Python

```bash
python password_generator_gui.py
Как пользоваться

Выберите длину пароля (рекомендуется 12–20)
Отметьте нужные типы символов:
Строчные буквы (a-z)
Заглавные буквы (A-Z)
Цифры (0-9)
Спецсимволы (!@#$...)

Нажмите одну из кнопок:
ПРОСТОЙ
СЛОЖНЫЙ
С НАСТРОЙКАМИ

Пароль появится в поле
Нажмите КОПИРОВАТЬ В БУФЕР

 Технические характеристики

Язык: Python 3.8+
Графический интерфейс: Tkinter
Сборка в EXE: PyInstaller (--onefile --noconsole)
Размер EXE: ≈ 8–12 МБ
Платформа: Windows 10/11 (64-bit)

Полностью открытый исходный код
